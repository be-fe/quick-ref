# avalon 

## 框架背景

去哪儿司徒正美牛整的一个能 IE 通吃的 MVVM 是图层框架.

## 与 ERP 的羁绊

在某个时期, ERP 有多个项目都启用了 avalon 作为基础框架. 

在历史上, 起了很大的作用, 但也成为我们团队历史项目里的包袱, 尤其是使用的 1.x (1.394) 版本, 文档较为缺失, 整体也不够完善, 导致在研发过程中有或多或少的一些阻碍.

除非 avalon 未来能证明自己, 成为主流, 否则团队不会再使用它进行新项目启动的基础.
